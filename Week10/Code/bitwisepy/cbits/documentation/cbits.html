Parameters

CBitCmp

Compares two CBits for implicit integral size. 

int CBitCmp(
    const CBit cb1,
    const CBit cb2);
Parameters
cb1

A CBit instance.

cb2

A CBit instance.

Return Value

0 if implied integral values are equal. Positive value if cb1 is larger than cb2. Negative value if cb2 is larger than cb1.

Discussion

This function can be passed to qsort and bsearch in stdlib.h
